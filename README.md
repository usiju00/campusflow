# CampusFlow

A Smart Student Identity and Transaction Management System built for Modibbo Adama University, Yola.

## Features
- Student registration and secure login
- JWT-signed QR code identity card
- Digital wallet (top-up and transfers)
- Transaction history
- Admin dashboard with analytics

## Tech Stack
- Python / Flask
- SQLAlchemy / SQLite
- Flask-Login / PyJWT / Bcrypt
- HTML5 / CSS3

## Modules
- Auth — register, login, logout
- Identity — QR code generation
- Wallet — top-up, transfer, history
- Admin — student and transaction analytics
